THE BOT

In Action

CALLING THE BOT

The bot can be configured to comment only when called. It's useful when spelling out the acronym is not always needed.

AUTOMATIC FUNCTION

The bot can be configured to automatically look for acronyms and comment. It's useful in a sub where there are a large number of acronyms, thus almost always requiring spelling out the acronym.

CONCISE & SINGLE COMMENT

The bot can be configured to make only one comment per post. Based on the trigger settings, any additional acronyms are edited into the same comment. It helps summarise the entire thing.

MULTIPLE COMMENTS

The bot can be configured to respond to every single comment based on trigger settings. It's useful for quick clarifications.
